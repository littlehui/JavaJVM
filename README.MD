# 简介

这是一个用Java写的JVM虚拟机，可以执行编译后的Class文件。参考 张秀宏的《自己动手写Java虚拟机》，可以作为此书的Java版代码来学习。

# 支持的特性

- [x] 控制台说明打印
- [x] 类路径扫描加载
- [x] JVM虚拟机的指令
- [x] 单方法执行
- [x] 类，方法调用
- [x] 数组
- [x] 字符串
- [ ] 自动拆箱装箱
- [ ] 本地方法
- [ ] 异常处理
